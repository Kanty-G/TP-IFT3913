Comme on a opté pour le format maven pour notre projet java, les 4 classes tloc.java, tassert.java, tls.java et tropComp.java
se trouvent dans un même paquet nommé metrics.

Pour compiler les 4 fichiers, on doit se rendre dans le répertoire metrics(tp1/src/main/java/metrics) et utiliser la commamde
javac filename

Pour l'execution,tloc.java et tassert.java avec "java -jar [chemin du fichier à évaluer]"  à partir de la ligne de commande ou
avec java -jar et rentrer le chemin du fichier java quand le programme demande de le demande

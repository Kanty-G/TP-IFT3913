Pour compiler notre code  il suffit d'utiliser la commande javac [nom du fichier java], en faisant attention d'être dans le repertoire TP1-IFT3913

Pour l'execution de tloc, tassert et tls on fait java -jar [nom du fichier jar] [chemin du fichier à évaluer].
Et pourtropComp il faut ajouter le nom des fichiers dans lesquels on veut stocker nos output en faisant: java -jar [chemin du fichier à évaluer] [chemin du fichier output]
seuil par exemple pour le seuil de 1% java -jar tropComp.jar ./etude-jfreeChart/output1.csv 1, pour le seuil 5, on va remplacer par output5.csv, et pour
le seuil 10 ça sera output10.csv
